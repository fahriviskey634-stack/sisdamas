# Glossary
## Platform Acronyms and Acronym References

This file defines the core abbreviations and terms used throughout the system.

### 1. Project Terms
*   **SISDAMAS:** Sistem Pemberdayaan Masyarakat (Community Empowerment System). The core methodology containing 4 cycles.
*   **KKN:** Kuliah Kerja Nyata. Student community service program spanning 40 days.
*   **LPJ:** Laporan Pertanggungjawaban. The official final report submitted to the campus. Supported by data exports.
*   **DPL:** Dosen Pembimbing Lapangan. Supervising lecturer who evaluates the KKN deliverables.
*   **Rembug Warga:** Cycle 1 community deliberation session to gather initial aspirations.
*   **USG Matrix:** Urgency, Seriousness, and Growth priority ranking method.
*   **PWA:** Progressive Web Application. Allows browser-based installation on Android phones with offline support.
*   **RLS:** Row Level Security. PostgreSQL feature to restrict row access at database query level.
*   **JWT:** JSON Web Token. Session credentials used for secure REST API communication.
*   **PII:** Personally Identifiable Information. Private resident names and card numbers that must be protected.
*   **OSM:** OpenStreetMap. Free base map tile provider.
*   **BaaS:** Backend-as-a-Service. In this project, Supabase provides DB, storage, and authentication.
*   **FAB:** Floating Action Button. Centered button on mobile bottom bar for quick entry.
*   **Idempotency Key:** Client-generated unique UUID used to identify requests and block duplicates.
