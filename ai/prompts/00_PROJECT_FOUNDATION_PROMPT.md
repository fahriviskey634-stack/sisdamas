# PROJECT BACKGROUND

This project is being developed to support the implementation of SISDAMAS (Sistem Pemberdayaan Masyarakat) during the Community Service Program (KKN) conducted by students from UIN Sunan Gunung Djati Bandung.

Current implementation is mostly manual.

Community aspirations are written on paper sticky notes.

Household surveys are recorded using paper or spreadsheets.

Documentation is scattered across WhatsApp, Google Drive, and personal devices.

Location data is manually copied into Google Earth or QGIS.

Program planning is discussed manually.

Reports are compiled manually.

This process is inefficient, difficult to monitor, and prone to data inconsistency.

The goal is to create one integrated digital platform that supports every stage of SISDAMAS.
