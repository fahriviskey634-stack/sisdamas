# USER ROLES

The platform only has three user roles.

## 1. Super Administrator

There is only ONE Super Administrator.

Responsibilities include:

- Full system administration
- User management
- Master data management
- RT/RW/Dusun management
- Category management
- Google Drive integration
- Google Calendar integration
- Database backup
- Restore data
- Lock / Unlock survey data
- View audit logs
- Export all data
- System configuration

The Super Administrator has unrestricted access.

---

## 2. KKN Team Member

All KKN members have exactly the same permissions.

Every member can:

- Login
- Fill survey forms
- Capture GPS coordinates
- Upload documentation
- Upload survey photos
- View dashboard
- View interactive map
- View statistics
- Create tasks
- Update task progress
- View reports
- Edit data they created
- Delete data they created
- Edit documentation they uploaded
- Delete documentation they uploaded

Members CANNOT:

- Edit other members' data
- Delete other members' data
- Manage users
- Change system configuration
- Access backups
- Access audit logs
- Modify master data

---

## 3. Public Visitor

No login required.

Visitors can only access:

- KKN Profile
- Village Profile
- Public Timeline
- Gallery
- Instagram
- TikTok
- Public statistics (if published)

Visitors cannot modify any data.
